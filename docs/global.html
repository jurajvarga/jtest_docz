assignToUser(manager, node, wfID, userID, isGroup, auditEventType, auditQueueMain, auditQueueApproved)

Assign task in the workflow specified by wfID to the provided user based on userID, build an audit message and send to audit queue.
Parameters:
Name Type Description
manager * STEP Manager
node * STEP Object (e.g. revision, product, figure folder, ...)
wfID String Workflow ID
userID String User ID
isGroup boolean indicates if the provided user is a group or a person
auditEventType * Derived Event Type for Audit
auditQueueMain * Audit outbound endpoint for Main Workspace
auditQueueApproved * Audit outbound endpoint for Approved Workspace
